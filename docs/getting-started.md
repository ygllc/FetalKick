# Getting started

Fetal Kick helps you record fetal movements with minimal setup.

Use it for simple, repeatable daily tracking.

{% hint style="warning" %}
Start with the plan from your healthcare provider. If movement feels reduced or unusual, contact them right away.
{% endhint %}

### Before you begin

* Use an Android device
* Pick a time you can track consistently
* Decide how you want to label sessions by meal

### Your first session

{% stepper %}
{% step %}
### Open the app

Launch `Fetal Kick` from your device.
{% endstep %}

{% step %}
### Start a new session

Tap `New Session` on the home screen.
{% endstep %}

{% step %}
### Choose meal and date

Select the meal label, then confirm the date.
{% endstep %}

{% step %}
### Count kicks

Tap the main button once for each kick.
{% endstep %}

{% step %}
### Exit to save

Leave the session when you are done.

Sessions with at least one kick save automatically.
{% endstep %}
{% endstepper %}

### What happens next

After saving, you can:

* review sessions in [History and export](history-and-export.md)
* change targets in [Settings, goals, and widget](settings-goals-and-widget.md)
* learn the full flow in [Track a session](track-a-session.md)

### Good habits

* Track around the same times each day
* Use the same meal labels consistently
* Export your data if you want a backup
