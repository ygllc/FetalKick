# Settings, goals, and widget

{% hint style="warning" %}
Settings and Export are currently in progress, so you won't see them yet.
{% endhint %}

Fetal Kick includes a few simple controls to match your routine.

### Goals

You can set a kick goal from `4` to `20`.

Use the goal that fits the plan from your healthcare provider.

### Session length

Set your preferred session duration if you use timed counting.

This helps you start a consistent timer faster.

### Home screen widget

The widget shows today’s total at a glance.

Use it when you want fast visibility without opening the app.

### Appearance

Fetal Kick supports:

* dark mode
* dynamic colors on Android `12+`

### Permissions

The app may ask for:

* **Storage** to export your data
* **Notifications** for optional reminders

Only grant the permissions you want to use.
