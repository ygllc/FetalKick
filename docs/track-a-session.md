# Track a session

Use a session each time you want to count movements.

### Start a session

{% stepper %}
{% step %}
### Open `Fetal Kick`.
{% endstep %}

{% step %}
### Tap the `New Session` FAB, aka.➕button.
{% endstep %}

{% step %}
### Choose a meal.
{% endstep %}

{% step %}
### Use the date picker and confirm with the OK button.
{% endstep %}

{% step %}
### Start counting your kicks!
{% endstep %}
{% endstepper %}

### Count kicks

Tap the main button once for each kick.

Your running count stays visible during the session.

### Use the timer

The timer is optional.

You can:

* start it
* pause it
* reset it

Use it when you want a fixed counting window.

### End the session

Tap `Exit` when you are done.

* If the count is greater than `0`, the session saves automatically.
* If the count is `0`, the session is discarded.

### Tips for consistent tracking

* Track around the same times each day.
* Use the same meal labels consistently.
* Count in a quiet place when possible.
* Export your data if you want to share trends.

See [History and export](history-and-export.md) next if you want to review saved sessions.
